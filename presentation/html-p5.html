JavaScript alapok

Osztályok

  • egyedi objektum létrehozására szolgáló minta

  • constructor függvény inicializálja
  • new utasítással hozunk létre új példányokat
  • meglévő osztályokból újakat származtathatunk az extends utasítással

JavaScript alapok

Modulok

  • más scriptekben felhasználható kódot tartalmazó JS fájlok
  • import utasítással használjuk
  • az importáló fájlt modulként kell betölteni HTML-ből
    <script type="module" … >
  • bármilyen kód kerülhet modulba, de jellemzően osztályokat és függvényeket tartalmaznak

p5.js

Alapfunkciók

  • inicializálás: setup()
  • rajzolás: draw()
  • vezérlés: loop(), noLoop(), redraw()
  • canvas létrehozása: createCanvas()

p5.js

Objektumok

  • 2D és 3D primitívek – point, line, circle, rect
  • görbék – curve, bezier
  • vertex – tetszőleges 2D vagy 3D pont,
    egyedi alakzatokhoz

p5.js

Koordináták és transzformáció

  • origó a canvas bal felső sarkában, jobbra és lefelé pozitív
  • translate(), rotate(), scale()
  • push() és pop() függvényekkel elmenthetjük ill. visszaállíthatjuk az aktuális transzformációt

p5.js

Grafikai attribútumok

  • fill: kitöltés színe
  • stroke: körvonal színe
  • strokeWeight: körvonal vastagsága
  • alpha: áttetszőség

p5.js

Szín

  • color()
    • colorMode(HSL) ajánlott
    • használható RGB, hex RGB, grayscale, color string
    • lerpColor() két szín közötti interpolációhoz

p5.js

Global vs. instance mode

  • alapértelmezett: global context (window object)
  • más JS könyvtárakkal vagy egynél több sketch használata esetén problémát okozhat
  • instance mode:
    • let sketch = (p5) => {
    p5.setup = () => {...};
    p5.draw = () => {...};
};
const app = new p5(sketch, 'elementId');
